## Overview

The software consists of the following modules:

- Home

  Data statistics and dashboard

- Mailbox Management

  Manage sender and receiver mailboxes

- Template Management

  Manage email body templates

- Sending Management

  Create new sends and view send history

- System Settings

  - User Management

    Add other user accounts

  - Basic Settings

    Global settings related to sending

  - Proxy Management

    Configure sending proxies

- Support the Author

  Sponsorship/donation page

- Help

  Documentation and help pages

## Sending Steps

1. Add sender accounts (skip if already added)
2. Add recipient lists (skip if already added)
3. Import required templates (skip if already added)
4. Open "New Send": enter subject -> select recipients -> select template -> select data -> preview to confirm send count and template correctness -> exit preview -> click "Send"
5. If sending fails, go to "Send History" to retry failed items
