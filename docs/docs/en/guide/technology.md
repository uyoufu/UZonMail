The system is developed using a frontend-backend separated architecture.

**Frontend:**

TypeScript + Vue 3 + Quasar

**Backend:**

C# + WPF + ASP.NET Core + WebView2

The server is implemented with ASP.NET Core, while the desktop client is developed with WPF.
