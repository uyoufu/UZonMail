This guide shows how to install UzonMail with Docker using the Baota (BT) control panel.

## Open terminal

![bt terminal](https://oss.uzoncloud.com:2234/public/files/images/image-20251018101402597.png)

## Start configuration

Follow the Docker installation steps in [Docker installation](/guide/installation/docker.html#安装步骤).

## Configure reverse proxy

After successfully starting the `uzon-mail` service, configure a reverse proxy in Baota via **Sites → Add Site**.

![add site](https://oss.uzoncloud.com:2234/public/files/images/image-20251020081730026.png)

After creating the site, click "Settings" to open the site configuration.

![site settings](https://oss.uzoncloud.com:2234/public/files/images/image-20251018104946594.png)
