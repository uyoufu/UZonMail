## Pricing

| Edition  | Annual     | Perpetual     |
| -------- | ---------- | ------------- |
| Open Source | Free   | Free          |
| Professional | ¥99/year | ¥299/perpetual |
| Enterprise  | ¥999/year | ¥2999/perpetual |

> As features continue to evolve and maintenance requires more effort, perpetual licenses may be withdrawn in the future.

## Note

::: tip

Most Enterprise edition features require server deployment and proxying to a domain.

:::
